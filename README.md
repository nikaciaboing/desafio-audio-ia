Gravação de Áudio, Transcrição e Resposta com IA

Aplicação web desenvolvida com Next.js que permite gravar áudio pelo navegador, realizar a transcrição automática utilizando inteligência artificial e gerar uma resposta com base no texto transcrito.

O projeto tem como objetivo demonstrar integração entre front-end moderno, APIs, manipulação de mídia e consumo de serviços de IA.

Tecnologias 
- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- OpenAI API
- MediaRecorder API

Funcionalidades
- Gravação de áudio diretamente no navegador
- Transcrição automática do áudio
- Geração de resposta por IA
- Interface simples e responsiva
- Feedback visual de carregamento
